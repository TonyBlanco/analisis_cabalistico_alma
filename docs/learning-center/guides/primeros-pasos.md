# Primeros pasos

## Objetivo

Esta guía te ayuda a orientarte en el dashboard del terapeuta y a reconocer dónde vive cada tipo de tarea.

## Flujo recomendado

1. Abre el menú del terapeuta.
2. Entra en **Aprender** para revisar el índice.
3. Recorre el tour si estás empezando.
4. Abre la guía del módulo que vayas a usar.

## Qué conviene recordar

- Este centro solo explica uso de la app.
- Cada guía enlaza con una pantalla real.
- Si no sabes dónde estás, vuelve al índice.
