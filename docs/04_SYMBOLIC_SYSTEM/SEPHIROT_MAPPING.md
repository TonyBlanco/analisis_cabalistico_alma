# Sephirot Mappings (PoC)

Simple mapping of Sephirot to symbolic attributes used by PoC.

Format:
- sephira_id: {
    'name_en': 'Keter',
    'planet': null,
    'chakra': 'crown',
    'color': 'white',
    'notes': 'Highest emanation - pure will'
  }

Example (minimal):
keter:
  name_en: 'Keter'
  planet: null
  chakra: 'crown'
  color: 'white'
  notes: 'Divine will, source'

netzach:
  name_en: 'Netzach'
  planet: 'venus'
  chakra: 'sacral'
  color: 'green'
  notes: 'Endurance, victory'

provenance:
source_name: 'Sefirot PoC mappings'
version: '0.1-poc'
curator: 'team'
