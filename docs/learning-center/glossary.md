# Glosario

## Aprender

Entrada del menú del terapeuta que abre el Centro de Aprendizaje.

## Guía

Documento canónico que explica el uso de un módulo o pantalla.

## Tour

Recorrido corto con pasos ordenados para ayudar al primer uso.

## Fallback

Respuesta segura que orienta al usuario cuando no hay una respuesta más precisa.
