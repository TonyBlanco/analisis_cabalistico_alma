# Resonancia Ancestral

## Para qué sirve

El workspace de Resonancia Ancestral te permite mapear y registrar resonancias entre generaciones del árbol genealógico del consultante. Una resonancia es un patrón que se repite a través del tiempo en el sistema familiar y que el terapeuta identifica y confirma manualmente. La app no genera resonancias automáticamente.

## Dónde está

Menú del terapeuta → **Resonancia Ancestral**
Ruta directa: `/dashboard/therapist/resonancia-ancestral`

## Tipos de resonancia que puedes registrar

El sistema contempla dos marcos de referencia:

**Armoni (numérico-sistémico)**
- Resonancia por número — mismos números de orden de nacimiento en distintas generaciones (grupos 1-4-7, 2-5-8, 3-6-9)
- Complejo del tío — patrón entre tíos y sobrinos
- Duplica generacional — repetición de un rol o destino
- Repetición de nombre — nombre compartido entre generaciones
- Síndrome de aniversario — eventos que ocurren en las mismas fechas
- Hueco por aborto o fallecido — impacto de una posición vacía en el sistema
- Proyecto u objetivo no logrado — que el sistema "hereda" a la siguiente generación
- Mismo rol en el sistema — distintas personas que ocupan la misma función

**Hellinger (sistémico-familiar)**
- Exclusión — alguien fue excluido del sistema y genera lealtad invisible
- Lealtad invisible — repetición inconsciente de un destino ajeno
- Asunto de orden — inversión del orden natural entre miembros

## Cómo registrar una resonancia

1. Selecciona el consultante activo.
2. Desde el workspace, usa el botón de nueva resonancia.
3. Elige el tipo y vincula las personas del árbol implicadas (persona A y persona B).
4. Indica la relevancia (alta / media / baja) y la dirección (bidireccional o unidireccional).
5. Añade notas observacionales si las tienes.
6. Guarda — la resonancia queda asociada al consultante y solo tú puedes modificarla.

## Captura por el consultante

Puedes habilitar que el consultante registre sus propias observaciones desde su portal. Esto se activa por consultante desde el workspace. Las observaciones del consultante son sugerencias — el terapeuta las revisa y confirma o descarta.

## Regla de uso

Ninguna resonancia se genera automáticamente. El terapeuta es el único autor que confirma resonancias. Esto protege la gobernanza clínica y evita interpretaciones mecanicistas del sistema familiar.
