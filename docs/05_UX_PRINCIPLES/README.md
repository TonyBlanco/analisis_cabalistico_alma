# UX Principles — Learning Center

Referencia rápida para las pantallas de ayuda y aprendizaje del terapeuta.

## Principios

- Mantener lenguaje operativo, nunca clínico.
- Priorizar jerarquía clara: índice, guía seleccionada, referencia, ayuda rápida.
- Usar tarjetas limpias, bordes suaves y contraste suficiente.
- Mantener una sola ayuda flotante por contexto para evitar ruido visual.
- Mostrar fallback visible cuando el grounding sea parcial o inexistente.
- Conservar accesibilidad básica: etiquetas, foco claro y navegación por teclado.

## Aplicación

Estas pautas se aplican al Centro de Aprendizaje, al widget de ayuda y a cualquier guía futura que se agregue bajo `docs/learning-center/`.
