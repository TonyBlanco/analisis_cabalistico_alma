# Centro de Aprendizaje

Este espacio explica cómo usar la app del terapeuta. No interpreta casos ni sustituye el criterio profesional.

## Qué encontrarás

- Onboarding y tour guiado
- Guías por módulo
- FAQ de uso
- Glosario operativo
- Novedades visibles de la interfaz

## Principio de uso

Trabaja siempre con el contexto correcto: primero el dashboard, después el workspace o la guía que corresponda.
