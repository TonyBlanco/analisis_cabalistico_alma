TESTS vs EXPLORACIONES — Documento de Distinción Legal y Ética

Estado: CANÓNICO — V1
Fecha: 2026-01-11

1. Propósito

Este documento establece de forma inequívoca la diferencia entre **tests clínicos** y **exploraciones simbólicas holísticas**, sus usos permitidos, límites legales y consideraciones éticas para el proyecto.

2. Qué ES / Qué NO ES
Qué ES

- Un documento normativo que clarifica intención, limitaciones y responsabilidades.
- Una guía legal y ética para el diseño y uso de exploraciones simbólicas.

Qué NO ES

- No es una justificación clínica para evaluaciones médicas.
- No habilita la publicación de resultados diagnósticos a usuarios.

3. Distinciones clave

- Propósito: Tests clínicos evalúan condiciones específicas; exploraciones simbólicas generan artefactos interpretativos para terapeutas.
- Visibilidad: Tests pueden generar scores clínicos; exploraciones NO muestran métricas al usuario.
- Terminología: Explícitamente PROHIBIDO usar términos diagnósticos o acrónimos clínicos dentro de exploraciones.

4. Legal y Ético

- Las exploraciones se consideran herramientas de acompañamiento y reflexión, no instrumentos de diagnóstico.
- El uso de las exploraciones DEBE estar respaldado por consentimiento informado que explique límites de visibilidad y alcance.
- Cualquier uso fuera de este marco DEBE ser reportado y evaluado por la gobernanza.

5. Recomendaciones de gobernanza

- Toda exploración DEBE incluir una declaración clara sobre su naturaleza no clínica.
- PROHIBIDO indexar exploraciones en repositorios clínicos o usarlas para estudios epidemiológicos sin aprobación legal.

Nota final de seguridad

La división entre tests y exploraciones es esencial para salvaguardar derechos, claridad ética y responsabilidad profesional.
