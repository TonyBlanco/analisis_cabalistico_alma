# Symbolic Overlay Viewer

Purpose
This module provides a read-only visual overlay for cross-system patterns.
It highlights tarot tokens, sefirot/paths, and event dates without interpretation.

Scope
- No AI execution
- No diagnosis or conclusions
- No chart rendering
- Unmounted module for future integration

Data sources
- Symbolic timeline events
- Cross patterns from rule-based analysis
- Patient context for labeling

Notes
This viewer only visualizes provenance-driven patterns. It does not infer or
explain meaning.
