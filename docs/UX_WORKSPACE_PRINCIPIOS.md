# UX_WORKSPACE_PRINCIPIOS.md

## Workspace terapéutico persistente

Este documento define los principios UX del workspace terapéutico
para mantener una experiencia clínica estable, no destructiva y
centrada en la observación humana.

---

## 1. Qué es un workspace persistente

- Un espacio de trabajo que permanece visible en todo momento.
- El terapeuta observa, consulta y anota sin perder contexto.
- Las utilidades se abren como paneles auxiliares, no como páginas.

---

## 2. Qué NO es

- No es un menú de navegación tradicional.
- No es un sistema de formularios encadenados.
- No es un flujo de diagnóstico ni automatización.

---

## 3. Reglas del sidebar

- Funciona como panel de control de utilidades.
- Acciones abren paneles, no rutas.
- Puede mostrar texto solo en hover o modo expandido.
- Permite múltiples herramientas activas.

---

## 4. Reglas de paneles

- Paneles no bloquean el workspace central.
- Se pueden colapsar, apilar y redimensionar.
- Mantienen el contexto visual del paciente.
- No obligan a cerrar ni abandonar la vista principal.

---

## 5. Regla de oro

El terapeuta observa antes de actuar.

Cualquier cambio que rompa este modelo se considera regresión UX.
