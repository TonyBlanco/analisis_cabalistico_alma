# Matriz de Dependencias Modulo x Modulo - Sistema Simbolico Holistico

## Principio fundamental
Ningun modulo simbolico es autosuficiente.
El diagnostico holistico solo emerge del cruce gobernado entre modulos.

## Tipos de dependencia
- D0 - Independiente: puede operar en modo observacional sin otros modulos.
- D1 - Dependencia blanda: mejora profundidad, no bloquea operacion.
- D2 - Dependencia dura: bloquea diagnostico si falta.
- D3 - Dependencia estructural: sin este modulo se pierde coherencia global.
