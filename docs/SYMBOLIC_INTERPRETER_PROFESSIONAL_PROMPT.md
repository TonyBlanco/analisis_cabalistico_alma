## 🔍 LAS 4 SECCIONES OBLIGATORIAS

### 1. Structural Panorama (`structural-analysis`)

**Objetivo**: Vista general arquitectónica del patrón

**Elementos a describir**:
- **Densidad estructural**: Alta/media/baja concentración de flujos
- **Énfasis vertical vs horizontal**: Predominancia de flujos ascendentes/descendentes vs laterales
- **Dominancia central vs lateral**: Activación de Tiferet/Yesod (columna central) vs Chesed-Gevurah / Netzach-Hod (laterales)
- **Concentración en triadas**:
  - Triada Supernal (Keter-Chokmah-Binah): Abstracto/Conceptual
  - Triada Ética (Chesed-Gevurah-Tiferet): Valores/Equilibrio
  - Triada Astral (Netzach-Hod-Yesod-Malchut): Manifestación

**Lenguaje esperado**:
```
"La estructura presenta 3 sefirot dominantes con 18 conexiones activas. 
Énfasis vertical predominante con flujos desde Keter hacia Tiferet. 
La densidad estructural indica un patrón complejo con concentración 
en la triada ética."
```

**NO permitido**:
- ❌ "La persona tiene mucha energía"
- ❌ "Esto indica un conflicto interno"
- ❌ "Debes trabajar en..."

---

### 2. Sefirotic Dynamics (`pattern-recognition`)

**Objetivo**: Análisis de relaciones entre Sefirot

**Elementos a identificar**:
- **Relaciones significativas**: Qué sefirot se conectan con mayor intensidad
- **Patrones por polaridad**:
  - Harmonic (verde): Expansión, resonancia
  - Integrative (naranja): Proceso de síntesis
  - Tensional (rojo): Restricción, límite
- **Balances estructurales**: Simetrías o asimetrías en columnas (Rigor-Misericordia-Equilibrio)
- **Imbalances**: Sobrecarga de una columna o triada

**Lenguaje esperado**:
```
"Los flujos se distribuyen en 8 armónicos, 6 integrativos y 4 tensionales. 
Predominancia de patrones armónicos entre Chesed-Tiferet-Netzach. 
La columna de Rigor (Binah-Gevurah-Hod) presenta menor activación 
que la columna de Misericordia, sugiriendo asimetría estructural."
```

**NO permitido**:
- ❌ "Tu personalidad es expansiva"
- ❌ "Tienes problemas con la autoridad"
- ❌ "Eres muy emocional"

---

### 3. Methodological Context (`educational-context`)

**Objetivo**: Explicación meta-metodológica

**Elementos a explicar**:
- **Qué enfatiza el método**:
  - Pitágoras: Reducción numerológica, números fundamentales
  - Gematría: Valor numérico hebreo
  - Milui: Deletreo completo de letras
  - Atbash: Cifrado inverso
  - etc.
- **Qué NO captura el método**:
  - Pitágoras NO incluye astrología
  - Gematría NO incluye posiciones planetarias
  - etc.
- **Cómo esto condiciona la estructura**:
  - Método matemático → patrones deterministas
  - Método simbólico puro → sin contexto temporal
  - etc.

**Lenguaje esperado**:
```
"Método aplicado: Gematría Estándar. Este método enfatiza el valor 
numérico de las letras hebreas según la tradición clásica. No captura 
dinámicas astrológicas ni contextos temporales. La estructura observada 
refleja exclusivamente la correspondencia numerológica del nombre analizado."
```

**Propósito educativo**:
- Entrenar al practitioner en **humildad metodológica**
- Recordar que cada método tiene **limitaciones inherentes**
- Evitar **fetichización del método** ("este método lo revela todo")

---

### 4. Professional Keys (`symbolic-comparison`)

**Objetivo**: Orientaciones para observación avanzada

**Elementos a proveer**:
- **Cues observacionales**: "¿Qué llama la atención estructuralmente?"
- **Preguntas exploratorias**: "¿Cómo se distribuye...?" "¿Qué ausencias...?"
- **Temas estructurales**: Ejes, triángulos, ausencias latentes

**Lenguaje esperado**:
```
"Para análisis profundo, considerar: ¿Qué sefirot dominantes establecen 
polaridades estructurales? ¿Cómo se distribuye la energía entre triadas? 
¿Qué ausencias latentes podrían ser significativas? La observación 
sistemática de flujos tensionales puede revelar zonas de potencial 
transformación estructural."
```

**NO permitido**:
- ❌ "Deberías enfocarte en Chesed"
- ❌ "Esto significa que tienes que..."
- ❌ "Te recomiendo trabajar..."

**SÍ permitido**:
- ✅ "Considerar la relación entre..."
- ✅ "Observar si la activación de X correlaciona con..."
- ✅ "Explorar la ausencia de flujos hacia..."

## 🆚 COMPARACIÓN: Versión Anterior vs Nueva

### Prompt Anterior (v1.0)
```
TAREA:
Genera 3-4 observaciones simbólicas educativas sobre este patrón estructural.
Cada observación debe tener:
1. Un título breve
2. Contenido descriptivo simbólico (2-3 frases)
```

**Problemas**:
- ❌ Demasiado abierto
- ❌ Sin estructura fija
- ❌ Número variable de observaciones (3-4)
- ❌ Tipos de observación no predecibles

### Prompt Nuevo (v1.1)
```
## OUTPUT STRUCTURE (MANDATORY)

Generate 4 observations following this EXACT structure:

### Observation 1: Structural Panorama
### Observation 2: Sefirotic Dynamics
### Observation 3: Methodological Context
### Observation 4: Professional Keys
```

**Ventajas**:
- ✅ Estructura fija de 4 observaciones
- ✅ Orden predecible
- ✅ Tipos específicos asignados
- ✅ Contenido guiado por sección
- ✅ Terminología profesional consistente

## 📊 FALLBACK MEJORADO

### Fallback Anterior (v1.0)
```
3 observaciones básicas:
- "Estructura básica identificada"
- "Flujos armónicos observados"
- "Contexto metodológico"
```

**Problema**: Muy genérico, poco útil.

### Fallback Nuevo (v1.1)
```
4 observaciones estructuradas con LÓGICA ALGORÍTMICA:

1. Panorama: Calcula dominancia, énfasis vertical, densidad
2. Dinámica: Cuenta polaridades (harmonic/integrative/tensional)
3. Contexto: Explica método aplicado y sus limitaciones
4. Claves: Genera preguntas exploratorias basadas en patrones
```

**Ventaja**: Aunque no hay IA, el fallback sigue siendo profesionalmente útil.

**Ejemplo de fallback**:
```
"La estructura presenta 2 sefirot dominantes y 3 presentes, con 12 conexiones 
activas. Distribución equilibrada entre ejes. La densidad estructural indica 
un patrón moderado. Los flujos se distribuyen en 5 armónicos, 4 integrativos 
y 3 tensionales. Predominancia de patrones armónicos..."
```

## 🧪 TESTING RECOMENDADO

### Test 1: Comparar AI vs Fallback
1. Ejecutar Pitágoras con paciente
2. Generar interpretación con IA
3. Desactivar IA
4. Generar fallback
5. Comparar las 4 secciones en ambos casos
6. Verificar que estructura es idéntica

### Test 2: Validar Lenguaje Profesional
1. Generar interpretación
2. Leer Observation 1 (Panorama)
3. Verificar mención de: densidad, énfasis vertical/horizontal, triadas
4. Verificar ausencia de: personalización, psicologización

### Test 3: Contexto Metodológico
1. Ejecutar diferentes métodos (Pitágoras, Gematría, Atbash)
2. Leer Observation 3 (Contexto)
3. Verificar que explica limitaciones específicas de cada método
4. Verificar lenguaje meta-metodológico

### Test 4: Claves Profesionales
1. Leer Observation 4 (Claves)
2. Verificar presencia de preguntas exploratorias
3. Verificar ausencia de consejos ("debes", "te recomiendo")
4. Verificar tono profesional (trainer-to-trainer)
