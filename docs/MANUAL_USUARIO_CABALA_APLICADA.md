## 1. INTRODUCCIÓN

### 1.1 ¿Qué es Cábala Aplicada?

Cábala Aplicada es un módulo de exploración simbólica que utiliza herramientas de la tradición cabalística como apoyo en contexto terapéutico. El módulo permite:

- Calcular valores numéricos de palabras (Gematría)
- Extraer acrósticos de textos (Notarikón)
- Permutar letras según sistemas tradicionales (Temurá)
- Visualizar mapas simbólicos en el Árbol de la Vida

**⚠️ IMPORTANTE:** Este módulo es una herramienta **OBSERVACIONAL**. No diagnostica, no predice, no sustituye el criterio profesional.

### 1.2 Principios Éticos

El uso de este módulo está regido por los siguientes principios:

1. **No-Interpretación Automática**: El sistema muestra datos, no conclusiones
2. **Soberanía del Terapeuta**: Toda interpretación corresponde al profesional
3. **Transparencia**: Todos los cálculos son auditables
4. **Respeto**: El simbolismo es un mapa, no una verdad absoluta
5. **Consentimiento**: El consultante debe conocer la naturaleza de la exploración

### 1.3 Alcance y Limitaciones

**Este módulo SÍ permite:**
- Cálculos numéricos precisos basados en tradición cabalística
- Visualización de patrones estructurales
- Exportación de datos para documentación
- Apoyo a la reflexión guiada

**Este módulo NO permite:**
- Diagnóstico psicológico o espiritual
- Predicciones de ningún tipo
- Interpretaciones automáticas
- Prescripciones terapéuticas

## 2. PRIMEROS PASOS

### 2.1 Acceder al Módulo

1. Inicie sesión como **Terapeuta**
2. Desde el Dashboard, seleccione un **Consultante**
3. Navegue a **Workspace Especializado → Cábala Aplicada**
4. Verá la interfaz principal con el Árbol de la Vida

### 2.2 Interfaz Principal

La interfaz se divide en tres áreas:

```
┌─────────────┬──────────────────────┬───────────────┐
│   Sidebar   │   Área Central       │  Panel Tools  │
│  (Métodos)  │   (Árbol de Vida)    │  (Historial)  │
└─────────────┴──────────────────────┴───────────────┘
```

- **Sidebar izquierda**: Selección de métodos cabalísticos
- **Área central**: Visualización del Árbol de la Vida
- **Panel derecho**: Historial, snapshots, interpretación, PDF

### 2.3 Seleccionar un Consultante

El módulo requiere un consultante activo para:
- Guardar cálculos en su historial
- Personalizar mapas con su fecha de nacimiento
- Generar exportaciones asociadas

Para cambiar de consultante, use el selector en la parte superior del dashboard.

## 3. MÉTODOS CABALÍSTICOS

### 3.1 Gematría

#### ¿Qué es?
La Gematría es el sistema de asignación de valores numéricos a letras hebreas. Cada letra tiene un valor fijo, y el valor de una palabra es la suma de sus letras.

#### Cómo usarla
1. Seleccione **Gematría** en el sidebar
2. Introduzca una palabra o frase en hebreo
3. Presione **Ejecutar**
4. Observe el valor numérico y palabras relacionadas

#### Valores de Letras (Sistema Estándar)
| Letra | Valor | Letra | Valor | Letra | Valor |
|-------|-------|-------|-------|-------|-------|
| א | 1  | י | 10 | ק | 100 |
| ב | 2  | כ | 20 | ר | 200 |
| ג | 3  | ל | 30 | ש | 300 |
| ד | 4  | מ | 40 | ת | 400 |
| ה | 5  | נ | 50 |   |     |
| ו | 6  | ס | 60 |   |     |
| ז | 7  | ע | 70 |   |     |
| ח | 8  | פ | 80 |   |     |
| ט | 9  | צ | 90 |   |     |

#### Ejemplo Práctico
**Palabra:** אהבה (ahavah - amor)  
**Cálculo:** א(1) + ה(5) + ב(2) + ה(5) = 13  
**Relación:** אחד (ejad - uno) también suma 13

#### Pautas de Interpretación
> La Gematría revela conexiones simbólicas entre conceptos. El terapeuta puede explorar con el consultante qué significan estas conexiones en su narrativa personal.

### 3.2 Notarikón

#### ¿Qué es?
El Notarikón extrae acrósticos de frases, formando nuevas palabras a partir de las iniciales, finales u otras posiciones de las letras.

#### Modos Disponibles
| Modo | Descripción |
|------|-------------|
| Primera letra | Extrae la primera letra de cada palabra |
| Última letra | Extrae la última letra de cada palabra |
| Alternando | Alterna entre primera y última |
| Letra central | Extrae la letra del medio de cada palabra |
| Solo vocales | Extrae solo vocales (א, ה, ו, י) |
| Solo consonantes | Extrae consonantes (resto de letras) |

#### Cómo usarlo
1. Seleccione **Notarikón** en el sidebar
2. Introduzca una frase en hebreo
3. Seleccione el **modo de extracción**
4. Presione **Ejecutar**
5. Observe el acróstico resultante y su Gematría

#### Ejemplo Práctico
**Frase:** בראשית ברא אלהים (Bereshit bara Elohim)  
**Modo:** Primera letra  
**Resultado:** בבא (valor = 5)

#### Pautas de Interpretación
> El Notarikón revela "mensajes ocultos" dentro de textos. Útil para explorar frases significativas del consultante o textos de sus narrativas.

### 3.3 Temurá

#### ¿Qué es?
La Temurá permuta letras según sistemas de sustitución predefinidos, transformando palabras en otras.

#### Sistemas de Permutación
| Sistema | Descripción |
|---------|-------------|
| Atbash | א↔ת, ב↔ש, etc. (inverso del alfabeto) |
| Albam | א↔ל, ב↔מ, etc. (mitades del alfabeto) |
| Avgad | Cada letra se desplaza una posición |

#### Cómo usarlo
1. Seleccione **Temurá** en el sidebar
2. Introduzca una palabra en hebreo
3. Seleccione el **sistema de permutación**
4. Presione **Ejecutar**
5. Compare original y transformado

#### Ejemplo Práctico
**Palabra:** בבל (Bavel - Babilonia)  
**Sistema:** Atbash  
**Resultado:** ששך (Sheshaj)

#### Pautas de Interpretación
> La Temurá revela "la otra cara" de una palabra. Útil para explorar polaridades y transformaciones en las narrativas del consultante.

## 4. EL ÁRBOL DE LA VIDA

### 4.1 Estructura

El Árbol de la Vida (Etz Chaim) es un diagrama simbólico con:
- **10 Sefirot**: Esferas de energía/conciencia
- **22 Senderos**: Conexiones entre Sefirot (letras hebreas)
- **3 Pilares**: Misericordia, Rigor, Equilibrio

```
            ┌─────┐
            │Keter│
            └──┬──┘
         ┌────┴────┐
     ┌───┴──┐  ┌───┴──┐
     │Chokmah│  │Binah │
     └───┬──┘  └───┬──┘
         │    ┌────┴────┐
         │┌───┴──┐  ┌───┴──┐
         ││Chesed│  │Gevurah│
         │└───┬──┘  └───┬──┘
         │    └────┬────┘
         │    ┌────┴────┐
         │    │ Tiferet │
         │    └────┬────┘
         │┌───┴──┐  ┌───┴──┐
         ││Netzach│  │ Hod  │
         │└───┬──┘  └───┬──┘
         │    └────┬────┘
         │    ┌────┴────┐
         │    │ Yesod  │
         │    └────┬────┘
            ┌────┴────┐
            │ Malkuth │
            └─────────┘
```

### 4.2 Las Diez Sefirot

| Sefirá | Significado | Atributo |
|--------|-------------|----------|
| Keter | Corona | Voluntad, conexión divina |
| Chokmah | Sabiduría | Intuición, chispa creativa |
| Binah | Entendimiento | Comprensión, análisis |
| Chesed | Misericordia | Amor, expansión |
| Gevurah | Rigor | Límites, disciplina |
| Tiferet | Belleza | Armonía, corazón |
| Netzach | Victoria | Persistencia, emoción |
| Hod | Esplendor | Mente, comunicación |
| Yesod | Fundamento | Identidad, sexualidad |
| Malkuth | Reino | Cuerpo, manifestación |

### 4.3 Lectura de Desequilibrios

El sistema mapea síntomas y patrones a Sefirot:

- **Ansiedad** → Hod (mente hiperactiva)
- **Depresión** → Tiferet (pérdida de centro)
- **Ira descontrolada** → Gevurah (exceso)
- **Falta de límites** → Chesed (exceso)

> **NOTA:** Estas correspondencias son **simbólicas**, no diagnósticas. El terapeuta guía la exploración.

## 5. RESULTADOS Y EXPORTACIÓN

### 5.1 Ver Historial

El panel derecho muestra:
- **Historial de sesión**: Cálculos de la sesión actual
- **Historial persistente**: Cálculos guardados en servidor

Para ver detalles de un cálculo, haga clic en él.

### 5.2 Exportar Cálculos

Cada resultado puede exportarse en tres formatos:

| Formato | Uso |
|---------|-----|
| JSON | Datos estructurados para procesamiento |
| TXT | Texto legible para documentación |
| PDF | Documento imprimible para expediente |

### 5.3 Holistic Export

El módulo se integra con el sistema **Holistic Export**:
- Exporta todos los datos cabalísticos del consultante
- Se incluye en el reporte holístico general
- Mantiene trazabilidad de todas las ejecuciones

## 6. CASOS DE USO TERAPÉUTICOS

### 6.1 Caso: Mapeo de Ansiedad

**Situación:** Consultante reporta ansiedad persistente.

**Procedimiento:**
1. Calcular Gematría de palabras clave del consultante
2. Observar correspondencias con Hod (mente) y Yesod (fundamento)
3. Explorar con el consultante las narrativas asociadas
4. Documentar patrones para seguimiento

**⚠️ Recordatorio:** NO diagnosticar ansiedad basándose en Gematría.

### 6.2 Caso: Exploración de Narrativas

**Situación:** Consultante trabaja con una frase significativa de su historia.

**Procedimiento:**
1. Aplicar Notarikón en diferentes modos
2. Calcular Gematría del acróstico resultante
3. Buscar palabras relacionadas numéricamente
4. Facilitar reflexión sobre conexiones simbólicas

### 6.3 Caso: Ciclos de Tikún

**Situación:** Consultante explora patrones recurrentes en su vida.

**Procedimiento:**
1. Calcular ciclo anual basado en fecha de nacimiento
2. Identificar Sefirá del ciclo actual
3. Explorar cualidades de esa Sefirá
4. Reflexionar sobre resonancias con experiencia actual

## 7. PREGUNTAS FRECUENTES

### ¿Es esto diagnóstico clínico?
**NO.** Este módulo es una herramienta de exploración simbólica. Nunca sustituye diagnóstico clínico profesional.

### ¿Los consultantes ven los resultados directamente?
Por defecto, solo el terapeuta ve los resultados. Si desea compartir, use la función de exportación y revise antes de entregar.

### ¿Cómo se relaciona con otros módulos SWM?
Cábala Aplicada se integra con:
- **SHA**: Auditoría de Armonía Sefirótica
- **Tarot**: Correspondencias con Arcanos
- **Transgeneracional**: Patrones heredados

### ¿Qué pasa con la IA?
La IA está disponible **solo para asistencia educativa**, nunca para interpretación de almas. Requiere aprobación explícita del terapeuta para cada uso.

### ¿Puedo usar texto en español?
Los métodos cabalísticos están diseñados para hebreo. Para español, use la transliteración o trabaje con traducciones.

### ¿Los cálculos se guardan automáticamente?
Los cálculos de sesión son temporales. Use el botón **Guardar Snapshot** para persistir en el servidor.

### ¿Cómo exporto todo el historial de un consultante?
Use **Holistic Export** desde el dashboard del consultante para obtener un reporte completo.

### ¿Puedo modificar los valores de Gematría?
No. Los valores siguen el sistema estándar (Mispar Gadol). Otros sistemas pueden agregarse en futuras versiones.

### ¿Qué hago si encuentro un error?
Reporte a soporte técnico con:
- Captura de pantalla
- Texto utilizado
- Método ejecutado
- Resultado esperado vs obtenido

### ¿Hay capacitación disponible?
Sí. Consulte con administración sobre talleres de formación en uso ético de herramientas simbólicas.
