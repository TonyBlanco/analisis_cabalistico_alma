# FAQ de uso

## ¿Dónde empiezo?

Empieza por el dashboard del terapeuta y luego abre **Aprender** si necesitas una guía concreta.

## ¿Puedo volver al tour?

Sí. El tour se puede reabrir, reanudar o descartar desde el propio Centro de Aprendizaje.

## ¿La ayuda rápida interpreta casos?

No. Solo explica cómo usar la interfaz y cómo encontrar el módulo correcto.

## ¿Qué hago si no veo lo que busco?

Vuelve al índice del centro y busca por nombre de módulo o categoría.
