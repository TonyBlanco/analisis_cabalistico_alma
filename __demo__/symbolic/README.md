# Symbolic Demo Harness

Purpose: Provide a controlled, visual-only demo for symbolic visualization.

This demo is not production, not clinical, and not diagnostic.

How to view:
- Manually import and render `__demo__/symbolic/DemoSymbolicViewer.tsx` in a local sandbox or dev page.
- Do not add routes or navigation for this demo.

Notes:
- Uses simulated PatientContext values.
- No persistence, no interpretation, no AI.
